
# OpenTenebris

**A Better Discord. A Private Social Platform. A Safe Community.**

## 🌑 What Is OpenTenebris?

OpenTenebris is a community-first social platform built on the principle that **you are not a product**. Unlike major platforms that log every click, track every IP, and sell your data to advertisers, OpenTenebris operates on a **privacy-by-default, surveillance-by-exception** model.

Logging of normal users: **NOTHING**.
We log threat actors: **EVERYTHING NEEDED TO STOP THEM**.

## 🚀 Getting Started

### For Users

1. Go to https://opentenebris.org
2. Create an account (no email required — use a burner if you want)
3. Join or create a community
4. Start chatting

### For Developers

**API Base URL:** `https://opentenebris.org/api`

```
# Example API call
curl -X GET https://opentenebris.org/api/health
```

**Authentication:** JWT tokens (obtained via platform login)

**Rate Limits:** Generous for normal users, strict for unauthenticated requests
